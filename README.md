## Installation

#### Web
```
$ C install
```

######  Run the application
```
$ npm start
```
######  Run the Dependencies 
```
$ npm install angular-route
$ npm install --global http-server
$ npm install --save ngx-mask
$ npm install --save sweetalert2
```
